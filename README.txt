Varas
=====

Varas is a python Pratt parser framework, which aims to be general purpose and easy to use.

Pratt parsers (also known as top down operator precendence parsers) are simple to use and resonably
efficent.  There are a few articles describing them on the web:

http://javascript.crockford.com/tdop/tdop.html
http://eli.thegreenplace.net/2010/01/02/top-down-operator-precedence-parsing/

Installation
------------

Install using python distutils:

  $ python setup.py install
